#  PhonePe Pulse Analytics Dashboard

##  Project Overview
This project presents an interactive **Power BI Dashboard** built on the 
PhonePe Pulse dataset to analyze digital payment trends across India. 
It provides insights into transaction volumes, user engagement, state-wise 
performance, and payment category distribution from 2018 to 2022.

---

##  Objectives
- Analyze digital payment transactions across different states of India
- Identify yearly growth trends in transaction amount and user counts
- Compare transaction activity between states and districts
- Study different transaction types (peer-to-peer, merchant payments, etc.)
- Support business decision-making through data-driven insights
- Visualize financial data using interactive Power BI dashboards

## 📈 Key Insights
-  **Total Transaction Amount:** 121.41 Trillion
-  **Total Transactions:** 72 Billion
-  **Total Users:** 3 Billion
-  **Average Transaction Value:** 1.69K

###  Top Performing States
1. Telangana
2. Maharashtra
3. Karnataka
4. Andhra Pradesh
5. Rajasthan

### Transaction Type Breakdown
- **Peer-to-Peer Payments** — Largest share (60.11%)
- **Financial Services**
- **Merchant Payments**
- **Recharge & Bill Payments**
- **Others**

### Yearly Growth Trend
- Continuous growth from **2018 to 2022**
- Significant acceleration after **2020** due to increased digital adoption
- Rapid rise in both transaction volume and user base post-COVID
